# Kindle to PDF 自動化ツール

PC版Kindleアプリからスクリーンショットを撮影し、Kindleの本をPDFファイルに変換するプロセスを自動化する、シンプルなデスクトップアプリケーションです。

## 主な機能

- **自動スクリーンショット:** 各ページを自動的にスクリーンショット撮影します。
- **ページめくり:** キー操作をシミュレートしてKindleアプリのページをめくります。
- **PDF変換:** 撮影したスクリーンショットを1つのPDFファイルにまとめます。
- **シンプルなGUI:** Tkinterで構築された、使いやすいインターフェース。
- **ウィンドウ自動検出:** Kindleアプリのウィンドウを自動的に見つけ、最前面に表示します。
- **フルスクリーンモード:** Kindleアプリをフルスクリーンモードにし、クリーンなキャプチャを実現します。

## ?? セキュリティに関する警告

このアプリケーションは、Kindleアプリケーションとの対話を自動化するために、あなたのマウスとキーボードを制御します。

- **スクリーンショット処理の実行中は、マウスやキーボードを操作しないでください。**
- このアプリケーションは画面全体のキャプチャを行うため、**機密情報が画面に表示されていないこと**を確認してください。

## 要件

- **OS:** Windows
- **Python:** 3.6以上
- **Kindle for PC:** Amazonの公式デスクトップアプリケーションがインストールされている必要があります。

---

## インストールとセットアップ

1.  **リポジトリをクローンします:**
    `ash
    git clone https://github.com/nishis2rp/kindle-to-pdf-app.git
    cd kindle-to-pdf-app
    `

2.  **Python仮想環境を作成し、有効化します:**
    `ash
    # 仮想環境の作成
    python -m venv venv

    # 有効化
    .\venv\Scripts\activate
    `

3.  **必要なライブラリをインストールします:**
    `ash
    pip install -r requirements.txt
    `

## 使用方法

1.  **Kindle for PC アプリケーションが実行中でないことを確認してください。** (アプリケーションが自動的に起動します)
2.  プロジェクトフォルダ内の dist ディレクトリにある KindleToPdf.exe ファイルをダブルクリックして起動します。
    (フルパスの例: C:\Users\shing\OneDrive\ドキュメント\kindle-to-pdf-app\dist\KindleToPdf.exe)
3.  アプリケーションウィンドウが表示されたら、キャプチャしたい総ページ数を入力します。
4.  **「Start」** ボタンをクリックします。
5.  スクリプトが自動的にKindle for PCを探して起動し、開いている書籍の最初のページに移動します。起動とページ移動に10?15秒ほどかかります。
6.  Kindleが起動し、最初のページに移動したら、アプリケーションウィンドウでキャプチャしたいページ数を入力して **「Start」** ボタンをクリックします。
7.  以降のプロセスは以前と同じです。スクリプトが自動的にKindleを全画面にし、スクリーンショットを撮影します。**処理中はマウスやキーボードを操作しないでください。**
8.  完了すると、成功メッセージが表示され、PDFが保存されている Kindle_PDFs フォルダを開くかどうか尋ねられます。

生成されたすべてのPDFは、実行可能ファイルと同じディレクトリ（またはそのサブディレクトリ）に作成される Kindle_PDFs ディレクトリに保存されます。
